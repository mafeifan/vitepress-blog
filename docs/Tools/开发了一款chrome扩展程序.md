# mafengwo-mp3-downloader

一款识别并下载马蜂窝游记页面背景mp3音乐的chrome扩展程序

## 主要功能

当打开游记页面 [比如](http://www.mafengwo.cn/i/8134474.html) 如果检测到有背景音乐，
会弹出包含歌曲信息的chrome桌面提醒，同时鼠标菜单右键有下载歌曲的选项。
